## Fase 4: Broken Link Building (30-90 dagen)

### Strategie
1. Vind supplement/gezondheid sites met dode links
2. Check of wij vergelijkbare/ betere content hebben
3. Mail de webmaster: "Je link naar [X] is dood. Wij hebben een vergelijkbaar artikel: [URL]"

### Tools
- Check My Links (Chrome extensie)
- Ahrefs broken link checker
- Dead Link Checker

## Outreach Email Templates

### Template 1: Guest Post Pitch (NL)
```
Onderwerp: Gastartikel voor [SITE] — [TOPIC]

Hallo [NAAM],

Ik schrijf je namens AmareNL, een platform over natuurlijke supplementen en darmgezondheid. We hebben een uitgebreid artikel geschreven over [TOPIC] dat perfect zou passen bij jullie lezers.

Het artikel is:
✅ 1500+ woorden, Nederlands
✅ Wetenschappelijk onderbouwd (PubMed referenties)
✅ Unieke content — niet eerder gepubliceerd
✅ Geen verkooppraatje — puur informatief

Zou je interesse hebben dit te publiceren? Ik stuur het graag ter review.

Groet,
Musa
AmareNL.com
```

### Template 2: Resource Page Outreach (NL)
```
Onderwerp: Suggestie voor [SITE] linkpagina

Hallo [NAAM],

Ik zag jullie linkpagina op [URL]. Wat een mooie collectie!

Wij hebben op AmareNL een uitgebreide gids over [TOPIC] — compleet met wetenschappelijke bronnen en praktische tips. Misschien een waardevolle toevoeging voor jullie bezoekers?

Link: https://amarenl.com/[URL]

Dank voor het overwegen!

Groet,
Musa
```

### Template 3: Broken Link (NL)
```
Onderwerp: Dode link gevonden op [SITE]

Hallo [NAAM],

Ik browsde jullie pagina [PAGINA] en merkte dat de link naar [DODE_LINK] niet meer werkt.

Wij hebben onlangs een artikel gepubliceerd over hetzelfde onderwerp: https://amarenl.com/[SLUG]

Misschien een goed alternatief voor de dode link?

Fijne dag!
Musa
```
